# DO_TASK – Task Reporting Dashboard

DO_TASK is a web-based task reporting and management application built using
React and modern frontend tools. The system supports role-based access for
Admins, Managers, and Employees, ensuring secure task assignment and reporting.

## Features

- Light / Dark theme toggle
- Role-based access control
  - Admin & Manager can assign tasks
  - Employees can only view and submit reports
- Secure role switching with re-login
- Task report submission with PDF upload
- Clean and responsive UI

## Tech Stack

- React (Vite)
- TypeScript
- Tailwind CSS
- shadcn/ui
